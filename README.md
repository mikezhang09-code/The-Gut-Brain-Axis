# The Gut-Brain Axis — An Interactive Map

An interactive, bilingual educational platform designed to visualize and decode the three main pathways communicating between our guts and brains (Vagal, Circulatory, and Immune).

## 🌟 Key Features

*   **Interactive Body Map**: Explore the anatomy with glowing, animated mechanisms displaying the specific paths and velocities of different gut-brain biological signals.
*   **30-Day Timeline Explorer**: A dynamic scrubber illustrating the temporal relationships between dietary/stress variations and downstream mood, inflammatory, or circulatory shifts.
*   **Scientific Deep Dives**: Drill down into cutting-edge peer-reviewed mechanisms regarding neuropod circuits, microbiome metabolization into SCFAs, and immune cascades.
*   **Bilingual Support**: Full application toggling between English and Simplified Chinese without page reloads.

## 🛠 Technologies Used

*   **Core**: HTML5, Vanilla JavaScript, CSS3
*   **Animation**: GSAP (GreenSock Animation Platform) + ScrollTrigger + DrawSVGPlugin
*   **Icons**: Lucide Icons
*   **Design Considerations**: Dark mode layout, glowing SVG animations via `feGaussianBlur`, CSS custom variables.

## 🚀 Running Locally

1.  Clone the repository:
    ```bash
    git clone https://github.com/mikezhang09-code/The-Gut-Brain-Axis.git
    ```
2.  Navigate to the directory:
    ```bash
    cd The-Gut-Brain-Axis
    ```
3.  Serve the files using a local server (e.g., `npx http-server` or VS Code Live Server).
4.  Open `index.html` in your modern browser.
